# 页面结构与跳转关系

## 菜单与模块入口

当前一级菜单如下：

- 综合态势
- 数据接入
- 数据治理
- 知识构建
- 智能检索
- 智能问答
- 分析预警
- 报告中心
- 系统管理

对应路由入口定义在 [src/router/index.ts](E:/codex-test/open-source-data-intel-platform/src/router/index.ts)，菜单配置定义在 [src/router/menu.ts](E:/codex-test/open-source-data-intel-platform/src/router/menu.ts)。

## 顶部全局入口

当前顶部导航已经预留并接入以下全局工作台入口：

- 统一检索 / 问答命令栏
- 专题视角切换（全域、经济、创新）
- 当前页面刷新入口
- 最近刷新时间显示

其中：

- 顶部“智能检索”提交 -> `/smart-search`，并携带 `keyword`、`mode=语义检索`、`topic`、`from=topbar-command`
- 顶部“智能问答”提交 -> `/smart-qa`，并携带 `question`、`autoAsk=true`、`topic`、`from=topbar-command`
- 顶部专题视角当前先作为全局上下文标识，后续在专题工作台阶段进一步落地
- 顶部刷新通过全局 `refreshToken` 触发当前视图重新挂载

## 首页联动关系

### 顶部统计卡片

- 数据源总数 -> 数据源管理
- 今日新增数据 -> 标准化数据列表，并带 `timePreset=today`
- 已治理数据量 -> 标准化数据列表，并带 `status=已标准化`
- 识别实体数量 -> 实体管理
- 告警数量 -> 预警记录
- 报告数量 -> 报告中心

### 地图图层联动

#### 综合图层

- 点击综合点位 -> 右侧面板展示对象详情
- 点击综合热点区域 -> 智能检索页
- 点击区域 -> 右侧区域统计

#### 预警图层

- 点击预警点位 -> 右侧面板显示预警摘要和关联对象
- 右侧“查看详情” -> 预警记录
- 右侧“查看关联事件” -> 事件管理

#### 科研图层

- 点击科研机构点位 -> 实体管理
- 点击科研事件点位 -> 事件管理
- 点击科研热点区域 -> 智能检索页，并带科研相关关键词

#### 产业图层

- 点击企业 / 园区 / 项目点位 -> 实体管理
- 点击产业事件点位 -> 事件管理
- 点击热点区域 -> 智能检索页或事件管理页

### 首页图表

- 数据来源分布 -> 数据源管理，并带 `sourceType`
- 数据增长趋势 -> 标准化数据列表，并带 `date`
- 热点主题排行 -> 智能检索，并带 `keyword`
- 数据类型分布 -> 智能检索，并带 `dataType`

### 首页底部列表

- 最新预警 -> 预警记录详情定位
- 最近采集任务 -> 采集任务日志
- 热门专题 -> 智能检索结果
- 最近生成报告 -> 报告详情

## 列表页与详情页关系

### 数据源管理

- 标题 -> 数据源详情抽屉
- 查看详情按钮 -> 数据源详情抽屉

### 采集任务管理

- 标题 -> 日志弹窗
- 数据源名称 -> 数据源管理

### 标准化数据列表

- 标题 -> 内容详情抽屉
- 详情中的标签 -> 智能检索
- 详情中的实体 -> 实体管理

### 实体管理

- 实体名称 -> 实体详情抽屉
- 详情中的关联事件 -> 事件管理
- 详情中的关联内容 -> 标准化数据列表

### 事件管理

- 事件名称 -> 事件详情抽屉
- 详情中的涉及实体 -> 实体管理
- 详情中的关联内容 -> 标准化数据列表

### 预警记录

- 预警标题 -> 记录详情抽屉
- 来源对象 -> 智能检索
- 触发规则 -> 规则详情
- 详情中的关联内容 -> 标准化数据列表

### 报告中心

- 报告名称 -> 报告详情
- 报告详情中的重点事件 -> 事件管理
- 报告详情中的相关实体 -> 实体管理
- 报告详情中的引用内容 -> 标准化数据列表
- 报告趋势图点击 -> 智能检索

## 智能检索联动

- 结果标题 -> 结果详情抽屉
- 标签 -> 当前检索页标签筛选
- 来源 -> 当前检索页来源筛选
- 结果中的相关实体 -> 实体管理
- 详情中的相关事件 -> 事件管理
- 详情中的继续检索关键词 -> 当前检索页再次发起检索
- 详情中的“查看标准化内容详情” -> 标准化数据列表

## 智能问答联动

- 历史问题 -> 切换回答
- 推荐问题 -> 直接提问
- 依据来源 -> 标准化数据列表
- 相关实体 -> 实体管理
- 支持通过 query 参数传入问题进行预填或自动提问

## 预警、实体、事件、内容、报告之间的闭环

- 预警记录可进入标准化内容、规则、检索结果
- 内容详情可进入实体
- 实体详情可进入事件和内容
- 事件详情可进入实体和内容
- 报告详情可进入事件、实体和内容
- 首页地图和图表可把用户重新送回检索、预警、实体、事件和报告

## query 参数约定

当前系统大量页面联动依赖 query 参数，常见参数包括：

- `keyword`
- `question`
- `type` / `contentType`
- `status`
- `source`
- `date`
- `region`
- `topic`
- `level` / `riskLevel`
- `autoOpen=first`
- `autoOpenLog=first`
- `autoAsk=true`
- `timePreset=today`
- `from`

这些参数是当前 mock 联动的核心基础，后续接真实接口时建议继续保留。
