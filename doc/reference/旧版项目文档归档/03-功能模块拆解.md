# 功能模块拆解

## 顶部统一工作台入口

### 模块目标

作为全站一级页面之上的统一入口，承接检索、问答、专题上下文切换和页面刷新状态展示。

### 主要页面

- 主布局顶部导航栏

### 主要功能点

- 智能检索 / 智能问答模式切换
- 顶部统一输入入口
- 推荐问题快捷触发
- 专题视角切换
- 当前页面刷新入口
- 最近刷新时间展示

### 当前已实现

- 顶部统一命令栏
- 基于 query 的检索 / 问答跳转
- 专题上下文切换
- 当前视图 refresh token 刷新机制

### 当前仍为 mock 的内容

- 全局统一搜索网关
- 真实问答上下文注入
- `/api/config/ui` 远端 UI 配置

### 后续可扩展方向

- 接真实智能入口网关
- 与专题工作台、事件中心、告警中心联动
- 接真实数据健康状态与刷新提示

### 与其他模块关系

该入口位于所有业务页面之上，是首页工作台、智能检索、智能问答和后续专题工作台的公共上层入口。

## 综合态势首页

### 模块目标

作为平台总览入口，展示统计概况、空间态势、辅助分析图表和业务列表，并把用户引导到检索、问答、预警、报告等模块。

### 主要页面

- 首页综合态势页

### 主要功能点

- 统计卡片
- 地图总览
- 快捷入口
- 最近关注
- 数据来源、趋势、主题、类型图表
- 预警、任务、专题、报告列表

### 当前已实现

- 卡片可点击跳转
- 地图图层切换
- 点位与区域联动
- 地图右侧面板
- 图表点击 drill down
- 底部列表联动

### 当前仍为 mock 的内容

- 地图区域和点位
- 指标数据
- 右侧对象详情

### 后续可扩展方向

- 接真实 GIS / GeoJSON
- 接真实热力/聚合服务
- 接实时告警流

### 与其他模块关系

首页直接联动检索、问答、预警、事件、实体、报告、数据治理和数据接入。

## 地图总览模块

把平台从普通看板升级为带空间态势感知能力的综合总览平台。

### 主要页面

- 首页中的地图总览组件

### 主要功能点

- 综合 / 预警 / 科研 / 产业图层切换

- 区域点击
- 右侧联动面板

### 当前已实现

- ECharts + SVG 地图
- 区域热度高亮
- 热点区域强调
- 点位 tooltip 和详情联动

### 当前仍为 mock 的内容

- 全部地图数据

- 点位关联对象

### 后续可扩展方向

- 聚合点
- 热力层
- 区域统计联动到真实列表
- GIS 服务接入

### 与其他模块关系

地图是首页进入预警、事件、实体、检索和报告的关键入口。
