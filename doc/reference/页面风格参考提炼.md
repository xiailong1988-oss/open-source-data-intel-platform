# 页面风格参考提炼

## 1. 要吸收的设计语言

### 深色主框架
- 外层框架采用更深的主背景
- 顶栏和侧栏形成控制台感
- 主体面板通过边框、发光和层级来区分，而不是只靠白底

### 主视图优先
- 首页应让地图或主要分析视图成为视觉中心
- 列表和说明区是辅助，不应抢主视图权重

### 模块化信息面板
- 面板之间要有统一的边界、留白和层级
- 同级面板不应出现高低错乱或边距失衡

### 高分辨率下的信息节奏
- 大屏不意味着无限拉伸内容
- 需要通过最大内容宽度、分栏比例和留白控制节奏

## 2. 本项目当前应转化出的具体表现

- 左侧导航更像情报工作台导航，而不是默认后台菜单
- 顶部栏更像统一操作台，而不是简单信息条
- 首页形成“指标区 + 主视图区 + 辅助分析区 + 业务追踪区”
- 列表页形成“页头 + 筛选区 + 表格主体 + 详情抽屉”的稳定结构
- 图表、地图、详情面板都使用同一套面板语言
