# 参考网站说明

## 本轮整改参考站点

### 1. World Monitor
- 链接：https://www.worldmonitor.app/
- 重点参考：
  - 地图作为主视图的排布逻辑
  - 深色控制台式框架
  - 左侧导航与右侧信息区的节奏控制
  - 面板之间的视觉层次与边界表达

### 2. Crucix
- 链接：https://www.crucix.live/
- 重点参考：
  - 高信息密度页面的节奏感
  - 模块化面板拼接方式
  - 大屏下留白与信息组织方式
  - 深色背景中的重点信息提亮方式

## 使用原则

本项目不是直接复制参考站点，而是吸收它们的：

- 布局逻辑
- 信息优先级
- 深色主题层级关系
- 地图 / 主视图 / 侧面板的产品语言
