# 15-配置说明

## 一、文档目标
本文档用于说明当前后端配置文件结构、环境变量、默认值和各环境差异，方便本地开发、联调和部署时统一配置口径。

---

## 二、配置文件清单
当前主要配置文件位于 `backend/src/main/resources/`：

1. `application.yml`
2. `application-dev.yml`
3. `application-prod.yml`
4. `application-test.yml`
5. `logback-spring.xml`

---

## 三、各配置文件职责
### 3.1 `application.yml`
公共配置，主要包括：
1. 应用名称
2. 默认激活环境
3. Flyway 公共参数
4. Jackson 时间与格式
5. MyBatis-Plus 逻辑删除
6. SpringDoc / Knife4j 路径
7. Actuator 暴露策略
8. 日志基础级别

### 3.2 `application-dev.yml`
本地开发环境配置，主要包括：
1. 本地 MySQL 数据源
2. 连接池基础参数
3. 数据库健康检查开启
4. 业务日志级别更详细

### 3.3 `application-prod.yml`
部署环境配置，主要包括：
1. 生产数据库连接
2. 连接池基础参数
3. 数据库健康检查开启

### 3.4 `application-test.yml`
测试环境配置，主要包括：
1. 关闭数据源自动配置
2. 关闭 Flyway 自动迁移
3. 让单元测试不依赖真实 MySQL

### 3.5 `logback-spring.xml`
日志输出配置，当前主要使用控制台输出，字符集为 UTF-8。

---

## 四、关键环境变量说明
| 环境变量 | 默认值 | 说明 |
| --- | --- | --- |
| `SPRING_PROFILES_ACTIVE` | `dev` | 当前激活环境 |
| `SERVER_PORT` | `8080` | 服务端口 |
| `DB_URL` | `jdbc:mysql://127.0.0.1:3306/haidian_intel_platform?...` | 数据库连接串 |
| `DB_USERNAME` | `root` | 数据库用户名 |
| `DB_PASSWORD` | 空 | 数据库密码 |
| `DB_MAX_POOL_SIZE` | `10` | 最大连接池大小 |
| `DB_MIN_IDLE` | `2` | 最小空闲连接数 |
| `FLYWAY_ENABLED` | `true` | 是否启用 Flyway |

---

## 五、当前核心配置说明
### 5.1 环境选择
默认使用：

```bash
SPRING_PROFILES_ACTIVE=dev
```

当前建议：
1. 本地开发与联调使用 `dev`
2. 部署环境使用 `prod`
3. 单元测试使用 `test`

### 5.2 数据库配置
当前工程使用 MySQL 8.x，默认数据库名：

```text
haidian_intel_platform
```

Flyway 默认会在应用启动时自动执行迁移脚本。

### 5.3 文档配置
当前接口文档默认开启，可访问：
1. `/doc.html`
2. `/v3/api-docs`

### 5.4 日志配置
当前日志默认输出到控制台，适合开发和基础部署环境。

说明：
1. 当前未配置滚动文件日志
2. 若后续部署要求提升，可再补充文件日志策略

---

## 六、推荐本地配置方式
本地开发建议通过环境变量覆盖数据库配置，例如：

```bash
set SPRING_PROFILES_ACTIVE=dev
set DB_URL=jdbc:mysql://127.0.0.1:3306/haidian_intel_platform?useUnicode=true^&serverTimezone=Asia/Shanghai^&allowPublicKeyRetrieval=true^&useSSL=false
set DB_USERNAME=root
set DB_PASSWORD=your-password
```

---

## 七、当前配置边界
当前配置体系明确不包含：
1. Redis
2. MQ
3. 对象存储
4. 第三方统一认证
5. 分布式配置中心

---

## 八、后续调整建议
1. 若准备进入正式环境，建议补充敏感配置统一管理方式。
2. 若准备接入真实日志平台，建议增加文件日志和 traceId。
3. 若准备支持多实例部署，建议补充更稳定的 token 存储方案。
