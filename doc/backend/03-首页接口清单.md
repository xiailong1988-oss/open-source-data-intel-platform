# 03-首页接口清单

## 一、文档目的
本文档汇总当前已经完成、可直接支撑首页联调的后端接口。

当前覆盖范围：
1. Task-03：地区与地图基础接口
2. Task-05：首页快讯条与主情报流接口
3. Task-06：地图点位、首页概览、地区态势摘要接口
4. Task-07：热点专题、重点关注对象接口
5. Task-11：首页联调演示数据初始化与联调说明补齐

所有接口统一返回：

```json
{
  "code": 0,
  "message": "success",
  "data": {},
  "timestamp": "2026-04-08T10:00:00"
}
```

## 三、接口清单
说明：
1. 以下首页接口当前保持公开访问，不要求登录。
2. 联调演示数据已由 `V10__init_demo_data_for_integration.sql` 预置。

### 3.1 地区树
`GET /api/regions/tree`

### 3.2 地区详情
`GET /api/regions/{regionId}`

### 3.3 地区边界 GeoJSON
`GET /api/regions/{regionId}/geo`

### 3.4 地图图层配置
`GET /api/map/layers`

### 3.5 首页快讯条
`GET /api/intel/ticker`

### 3.6 首页主情报流
`GET /api/intel/stream`

### 3.7 首页地图点位
`GET /api/map/points`

支持参数：
- `regionId`
- `eventTypeId`
- `startTime`
- `endTime`

### 3.8 首页概览统计
`GET /api/dashboard/overview`

支持参数：
- `regionId`
- `eventTypeId`
- `startTime`
- `endTime`

### 3.9 地区态势摘要
`GET /api/dashboard/region-summary`

支持参数：
- `regionId`
- `eventTypeId`
- `startTime`
- `endTime`

### 3.10 首页热点专题
`GET /api/dashboard/hotspots`

支持参数：
- `regionId`
- `limit`

返回字段：
- `id`
- `title`
- `category`
- `regionId`
- `regionName`
- `heat`
- `mentions`
- `updatedAt`
- `department`
- `keyword`
- `summary`
- `detailId`

说明：
1. 默认只返回启用中的专题数据
2. 默认按 `heat desc, updatedAt desc` 排序
3. 当前返回结构适合首页热点榜、热点卡片直接绑定

### 3.11 首页重点关注对象
`GET /api/dashboard/focus-targets`

支持参数：
- `regionId`
- `limit`

返回字段：
- `id`
- `title`
- `targetType`
- `regionId`
- `regionName`
- `level`
- `levelLabel`
- `latestEventTime`
- `latestEventTitle`
- `status`
- `keyword`
- `summary`
- `detailId`

说明：
1. 默认只返回启用中的重点关注对象
2. 默认按 `level desc, latestEventTime desc` 排序
3. 当前返回结构适合首页重点关注列表、对象卡片直接绑定

7. 首页主情报流
8. 首页热点专题
9. 首页重点关注对象

---

## 五、前端接入建议
1. 地图区块先调 `GET /api/regions/tree`、`GET /api/regions/{regionId}`、`GET /api/map/points`
2. 顶部统计区块调 `GET /api/dashboard/overview`
3. 当前地区摘要调 `GET /api/dashboard/region-summary`
4. 热点专题区块调 `GET /api/dashboard/hotspots`
5. 重点关注区块调 `GET /api/dashboard/focus-targets`
6. 快讯条和主情报流继续调 `GET /api/intel/ticker`、`GET /api/intel/stream`
7. 建议优先使用 `regionId=3001`、`3002`、`3004`、`3007` 做首页演示验证
