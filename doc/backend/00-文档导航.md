# 00-文档导航

## 一、文档目标
本文档用于给当前 `/doc/backend/` 文档做统一导航，方便后续交付、接手和继续迭代时快速定位资料。

---

## 二、基础总览
1. `01-后端工程结构说明.md`：说明工程模块结构、目录职责和当前边界。
2. `02-数据库详细设计.md`：说明 Flyway 版本、表结构和初始化数据范围。
3. `03-首页接口清单.md`：汇总首页公共接口清单。
4. `05-Codex分任务执行清单.md`：记录 Task 执行状态和阶段结果。

4. `09-热点专题与重点关注模块开发.md`
