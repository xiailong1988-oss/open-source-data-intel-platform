## 最新人工反馈覆盖说明
- 用户最新人工测试反馈明确指出：此前实现中把“轮播框”误解成了底部横向播放轴；
- 现已按最新反馈纠正为：右侧固定框体中的“竖向自动上滚情报流”；
- 因此，本轮首页结构调整不再把底部播放框作为主轮播承接层，而是改为由右侧情报流承担“风险 / 突发 / 热点 / 重点关注标题自动滚动”的主职责。
- 本轮继续依据最新人工反馈做第二次纠偏：
  - 首屏地图必须继续放大
  - 左右栏必须进一步压缩
  - 右侧情报流必须默认就能连续自动上滚，不能因为数据过少而看起来像静态块
- 本轮又进一步依据《基于参照网站的首页版式映射设计文档》执行：
  - 首页内部不再保留左侧大面积摘要区
  - 地图顶部改为窄横条摘要与图层轻控制
  - 右侧情报流作为参照站式解释层保留并强化

## 任务目标
基于已经确认的《综合态势首页重构详细设计任务书》，在不推翻现有首页驾驶舱代码结构的前提下，完成首页从“已有地图的后台首页”向“地图主导型情报驾驶舱”的结构性重构。

本轮任务同时包含两部分：
1. 完成待执行任务目录清理与归档；
2. 按分阶段方式执行首页重构，避免一次性修改过大导致超时或引入大范围回归问题。
