# 任务：首页第二阶段重构 - 左中右下驾驶舱增强

## 任务状态
- 状态：已完成，待人工验收
- 开始时间：2026-04-01
- 最后更新时间：2026-04-01
- 任务性质：在当前“地图主舞台 + 右侧 rail”基础上进行第二阶段增强，不推翻方向，但补齐围绕式信息层与现代交互动效

## 任务背景
- 当前首页已经修正到“中央地图 + 右侧 signal rail”的基础关系，且桌面端左右骨架已经稳定。
- 但用户最新反馈明确指出：首页仍存在以下问题：
  1. 信息量偏少；
  2. 地图占比偏大，其他重要信息不够突出；
  3. 整体科技感与新意不足；
  4. 交互偏传统，缺少现代前端动效与沉浸感；
  5. 首页目前只有“地图 + 右侧 rail”，缺少左侧和底部对地图的围绕式支撑。

## 本轮目标
把首页从“地图 + 右侧 signal rail”的第一阶段骨架，升级为：

```text
顶部轻 chrome
左侧态势脉冲区 + 中央地图主舞台 + 右侧 signal rail
底部热点信息带
```

核心要求：
1. 地图仍然是主舞台，但略缩小；
2. 左侧补出轻量态势信息区；
3. 底部补出辅助热点信息带；
4. 右侧 rail 保留但更纯粹、更有动感；
5. 引入现代前端交互动效技术，增强科技感。
